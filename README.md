# ThermalLabeler
ThermalLabeler — это утилита для печати этикеток на термопринтерах из Android без привязки к конкретному производителю и без фирменных драйверов.

Программа является мостом между Android и термопринтером, даёт полный контроль над печатью этикеток там, где стандартные решения не работают или мешают.

Программа решает практическую проблему: как напечатать этикетку с телефона или планшета на обычном термопринтере.

Перед печатью содержимое файла автоматически преобразуется в растровый макет, масштабируется под реальные размеры этикетки и подготавливается с учётом ориентации, отступов и параметров принтера.

ThermalLabeler принимает данные для печати, корректно масштабирует под реальные размеры этикетки и учитывает.
Шаблоны этикеток:
В программе можно создавать и сохранять несколько шаблонов этикеток
с разными размерами и параметрами печати.

Каждый шаблон содержит набор свойств, таких как:
- ширина и высота этикетки
- зазор между этикетками (gap)
- смещение
- ориентация и поворот
- параметры выравнивания
- препринт (непечатаемая область)

Созданные шаблоны используются повторно и позволяют быстро переключаться
между разными типами этикеток без повторной настройки.

Печатает напрямую:
- через Bluetooth
- через USB
- через Wi-Fi

Работает как Android PrintService, т.е.:
- доступна из стандартного диалога «Печать»
- может использоваться любыми приложениями (1С, браузер, PDF-viewer и т.п.)

Программа позволяет открывать и печатать этикетки из следующих типов файлов:
- PDF — документы и макеты, полученные из других приложений или систем
- HTML — страницы и шаблоны, включая автоматически сформированные отчёты и ценники
- Изображения — PNG, JPG и другие распространённые форматы

История печати:<br>
Программа сохраняет историю выполненных заданий печати.
Для каждого задания фиксируется исходный файл и параметры печати.
Из истории можно повторно открыть задание печати и выполнить повторную печать
без необходимости заново выбирать файл или настраивать параметры этикетки.

Чем она отличается от «обычных» решений:
- не эмуляция A4
- не привязка к одному бренду принтера
- точная печать этикетка в этикетку
- контроль всего пайплайна печати
- подходит для складов, магазинов, логистики, маркировки

Типовые сценарии использования:
- Печать ценников и штрихкодов
- Печать складских и транспортных этикеток
- Печать из 1С, самописных Android-приложений
