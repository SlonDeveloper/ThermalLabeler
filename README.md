# ThermalLabeler
ThermalLabeler — это утилита для печати этикеток на термопринтерах из Android без привязки к конкретному производителю и без фирменных драйверов.
Программа является мостом между Android и термопринтером, даёт полный контроль над печатью этикеток там, где стандартные решения не работают или мешают.
https://github.com/SlonDeveloper/ThermalLabeler/blob/main/README.md
Программа решает практическую проблему: как напечатать этикетку с телефона или планшета на обычном термопринтере.

ThermalLabeler Принимает данные для печати, корректно масштабирует под реальные размеры этикетки и учитывает:
- ширину и высоту этикетки
- зазор (gap)
- смещение
- поворот (rotation)

Печатает напрямую:
<ul>
  <li>через Bluetooth
  <li>через USB
  <li>через Wi-Fi
</ul>

Работает как Android PrintService, т.е.:
- доступна из стандартного диалога «Печать»
- может использоваться любыми приложениями (1С, браузер, PDF-viewer и т.п.)

Чем она отличается от «обычных» решений:
✔ не эмуляция A4
✔ не привязка к одному бренду принтера
✔ точная печать этикетка в этикетку
✔ контроль всего пайплайна печати
✔ подходит для складов, магазинов, логистики, маркировки

Типовые сценарии использования:
✔ Печать ценников и штрихкодов
✔ Печать складских и транспортных этикеток
✔ Печать из 1С, самописных Android-приложений
